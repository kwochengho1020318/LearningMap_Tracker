## Architecture
### Requirements
* Ollama installed  [Link](https://ollama.com/)
* PostgreSQl installed and create a database  [Link](https://www.postgresql.org/download/)
* Enable Google custom search API [Link](https://console.cloud.google.com/apis/dashboard?project=learningmaptracker)
* Configure environment variables in .env file in both lmt_service and lmt_web
* Run both project
### Technologies
* Ollama For running LLM(gemma 12b in the project)
* React-Frontend
* Express-Backend
## Contribute
